O que é isso?

É o portfólio de Fernando Aguiar. Um dia vai se transformar em algo maior, mas por enquanto, funciona apenas como um projeto pessoal para mim.

Eu sou um desenvolvedor autodidata interessado em desenvolvimento de software, web e jogos. Gosto de aprender e posso colaborar em várias áreas da indústria de tecnologia, como desenvolvimento web, desenvolvimento back-end e dados. Também gosto de escrever, ler e planejar meus projetos. Alguns deles foi construído em ambiente de sala de aula e outros por conta própria, através de estudo livre.

Estou formadondo em técnico de informática pelo Senac-DF. Neste espaço, há uma página que funciona como um blog , onde compartilho trechos de livros, músicas e outros escritos como forma de passar o tempo. Espero que vocês gostem, e isso seja útil de alguma forma.

Animais fantástico

Este projeto está sendo desenvolvido como parte do curso completo de JavaScript ES6+ da Origamid. O JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, amplamente utilizada no desenvolvimento de páginas web interativas e aplicativos web. Ele permite a criação de scripts que podem ser executados diretamente no navegador, tornando as páginas da web mais dinâmicas e interativas para o usuário final. O JavaScript também pode ser utilizado para desenvolver aplicativos do lado do servidor, por meio do Node.js. Além disso, é uma das linguagens de programação mais populares e amplamente utilizadas atualmente.

Calculadora

Este projeto foi desenvolvido na IDE Visual Studio 2022 utilizando o framework Xamarin e a linguagem de programação C#. O Xamarin é uma plataforma de software livre para a criação de aplicativos modernos e de alto desempenho para iOS, Android e Windows utilizando a plataforma .NET. Com o Xamarin, é possível compartilhar até 90% do código entre diferentes plataformas, o que torna o processo de desenvolvimento mais rápido e eficiente. Além disso, o Xamarin possui uma grande comunidade de desenvolvedores ativos e uma ampla variedade de recursos e ferramentas disponíveis para auxiliar no processo de criação de aplicativos móveis multiplataforma.
